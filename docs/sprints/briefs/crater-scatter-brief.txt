New crater structure — four elements only:

function CraterImpression({ cx, cy, radius, ringCount, fillColor, isColored, isDark }) {
  const rimColor = isDark ? '#2a2835' : '#d4cfc8'
  const pitColor = isDark ? '#0d0d18' : '#b8b0a6'
  const ringColor = isDark ? '#1e1c2a' : '#c8c2ba'
  const shadowColor = isDark ? 'rgba(0,0,0,0.5)' : 'rgba(100,90,80,0.25)'
 
  // The actual fill — gray in TERRAIN mode, team color in TEAMS mode
  const activeFill = isColored ? fillColor : pitColor

  return (
    <g>
      {/* 1. Cast shadow — soft ellipse offset below-right */}
      <ellipse
        cx={cx + radius * 0.15}
        cy={cy + radius * 0.2}
        rx={radius * 1.1}
        ry={radius * 0.4}
        fill={shadowColor}
        filter="url(#craterShadow)"
      />

      {/* 2. Outer rim — slightly raised lip */}
      <circle
        cx={cx} cy={cy} r={radius}
        fill={rimColor}
      />

      {/* 3. Depression — inward gradient from rim to pit */}
      <circle
        cx={cx} cy={cy} r={radius * 0.92}
        fill={`url(#depression-${cx}-${cy})`}
      />
      <radialGradient id={`depression-${cx}-${cy}`} cx="42%" cy="38%">
        <stop offset="0%" stopColor={rimColor} stopOpacity="0.6"/>
        <stop offset="40%" stopColor={activeFill} stopOpacity="0.8"/>
        <stop offset="100%" stopColor={activeFill} stopOpacity="1"/>
      </radialGradient>

      {/* 4. Concentric rings — subtle recessed lines */}
      {Array.from({ length: ringCount }).map((_, i) => (
        <circle
          key={i}
          cx={cx} cy={cy}
          r={radius * 0.85 * ((ringCount - i) / ringCount)}
          fill="none"
          stroke={ringColor}
          strokeWidth={0.8}
          strokeOpacity={0.6 - i * 0.1}
        />
      ))}

      {/* Center pit — darkest point */}
      <circle
        cx={cx} cy={cy} r={radius * 0.12}
        fill={isDark ? '#050508' : '#8a8078'}
        opacity={isColored ? 0.6 : 1}
      />
    </g>
  )
}

// Shared shadow filter in defs
<filter id="craterShadow" x="-20%" y="-20%" width="140%" height="140%">
  <feGaussianBlur stdDeviation="3" result="blur"/>
  <feOffset dx="2" dy="4" in="blur"/>
</filter>

Per-crater color resolution:

function getCraterFill(teamId: string, teamColor: string): {
  isColored: boolean,
  fillColor: string
} {
  // Locked team always colored
  if (lockedTeam === teamId)
    return { isColored: true, fillColor: teamColor }
 
  // Hovered team colored (only if nothing locked)
  if (!lockedTeam && hoveredTeam === teamId)
    return { isColored: true, fillColor: teamColor }
 
  // TEAMS mode — all colored, deemphasize non-focused
  if (colorMode === 'TEAMS') {
    const isDeemphasized = (lockedTeam || hoveredTeam) &&
      teamId !== (lockedTeam ?? hoveredTeam)
    return {
      isColored: true,
      fillColor: isDeemphasized
        ? (isDark ? '#2a2835' : '#c8c2ba')  // pottery gray
        : teamColor
    }
  }
 
  // Default TERRAIN — no color
  return { isColored: false, fillColor: '' }
}

Would it be cool to have the short labels like ‘ATL’ also be recessed in the graph or written on like text for contrast against the texture?

Yes — and that’s actually the detail that would push it from “nice visualization” to “coherent physical object.” Right now labels float above craters as screen text, which breaks the material illusion the moment your eye hits them.
There are two directions that both work, and they feel quite different:
Recessed / embossed text — the letters are pressed INTO the surface like a stamp or die-cut. This is the most consistent with the Horseman aesthetic since everything in that reference is a relief effect. Technically it’s achieved with a duplicate text element offset 1-2px down-right in a darker color (the shadow cast by the depression) and the main text in a lighter color (the raised edge catching light). No actual 3D — just two text layers creating the illusion of depth. The result feels like text machined into metal or pressed into clay.
Handwritten / scratched text — thin, slightly irregular letterforms that look like they were scribed into the surface with a stylus. You’d use a thin stroke weight on a condensed monospace font, maybe with a very subtle SVG feTurbulence displacement filter applied to the text path to give it micro-irregularity. This is harder to execute cleanly but has a more organic, tactile quality.
The recessed approach is the right call for your dashboard — it’s consistent with your existing neumorphic language (which is fundamentally about inset/outset shadows creating depth), it’s achievable cleanly in SVG, and it matches the Horseman reference most directly.
Here’s the exact SVG implementation:

function RecessedLabel({ x, y, text, isDark }) {
  const shadowColor = isDark
    ? 'rgba(0,0,0,0.8)'
    : 'rgba(80,70,60,0.45)'
  const highlightColor = isDark
    ? 'rgba(255,255,255,0.12)'
    : 'rgba(255,255,255,0.9)'
  const baseColor = isDark
    ? 'rgba(140,130,120,0.6)'
    : 'rgba(110,100,90,0.7)'

  const style = {
    fontFamily: 'JetBrains Mono',
    fontSize: 9,
    fontWeight: 700,
    letterSpacing: '0.12em',
    textAnchor: 'middle' as const,
  }

  return (
    <g>
      {/* Shadow layer — offset down-right, dark */}
      {/* Creates the illusion of a depression casting shadow */}
      <text
        x={x + 0.8} y={y + 1.0}
        fill={shadowColor}
        {...style}
      >
        {text}
      </text>

      {/* Highlight layer — offset up-left, light */}
      {/* Creates the illusion of a raised edge catching light */}
      <text
        x={x - 0.5} y={y - 0.5}
        fill={highlightColor}
        {...style}
      >
        {text}
      </text>

      {/* Base layer — the actual letterform color */}
      <text
        x={x} y={y}
        fill={baseColor}
        {...style}
      >
        {text}
      </text>
    </g>
  )
}


Three text layers, two pixel offsets. The shadow offset (0.8px right, 1.0px down) is slightly larger than the highlight offset (0.5px left, 0.5px up) because real recessed geometry casts more shadow than it catches highlight — that asymmetry is what makes it read as genuinely pressed in rather than raised out.
One additional touch worth adding
Place the label inside the crater depression rather than above it. Center it at cy + craterRadius * 0.55 — sitting in the lower half of the bowl. This does two things: it removes the floating label above the crater which breaks the physical metaphor, and it makes the text feel like it was stamped into the bottom of the impression. Smaller craters (lower PPG gap) have less room but at 9px monospace font even an 18px radius crater has enough space for 3-4 characters.
The combined effect — recessed text sitting inside a pressed crater on a textured plaster surface — is exactly the Horseman aesthetic. Every element is part of the same physical material rather than data elements layered on top of a background.
