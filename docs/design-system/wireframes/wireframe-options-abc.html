Options A, B, C — Three-Zone Architecture Exploration — Sprint 13

Three-Zone Architecture — rightAction Slot

Zone 1 — Data & View Controls (Left)

Filters (Conference), Metrics (PPG/GD), Symbology (H/A vs Team), Perspective (Index vs Components). Uses SegmentedControl primitives.

Zone 2 — Analysis Actions (Center)

Trendline/average overlays, CardInsightToggle (AI lightbulb). Layers analytical context over the existing view.

Zone 3 — Utility Actions (Right)

Methods toggle (FlaskConical), MaximizeButton (Expand). Universal actions related to the chart container itself.

Option A Inline Three-Zone (Single Row)

All three zones sit in a single flex row to the right of the title, separated by subtle vertical dividers. Most compact option — works best for charts with 1-2 segmented controls.
