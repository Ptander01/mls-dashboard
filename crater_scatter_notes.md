# Crater Scatter Reference Notes

From the user's reference images:
- Concentric rings pressed into matte clay/plaster surface
- Larger values = deeper craters with more concentric rings
- Small values = tiny dimples or single-ring impressions
- Connected by dotted/dashed lines (like constellation paths)
- Monochromatic matte finish - no gloss
- Depth creates natural shadow on one side, highlight on the other
- Ring spacing is even, creating a topographic contour effect
- The overall effect is like a lunar surface or impact map

Design translation for SVG:
- Each data point becomes a "crater" with concentric rings
- Number of rings = function of squad depth (more depth = more rings)
- Crater diameter = function of PPG gap magnitude
- Depth illusion via: inner shadow gradient, ring stroke with varying opacity
- Subtle cast shadow beneath each crater
- Matte surface texture via subtle noise or grain overlay
- Team color tints the crater subtly
- Dotted regression line connects through the field like constellation paths
